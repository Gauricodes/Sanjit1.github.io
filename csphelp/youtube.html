Adding a Youtube Video to your website

To add a youtube video you will first need to go to youtube, and where the like and dislike buttons are click share. One of the first options should be Embed with a < > logo(Hint Hint they are teasing html code). Then customize the embed link by playing with the options, and then click copy. It should give you an HTML tag like this: <iframe width="560" height="315" src="https://www.youtube.com/embed/wLoYIBEZEfw?start=240" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

THis tag is called an iframe (duh). The width="" defines width of course. height defines the height of the iframe, and src="" defines the link to be embedded.
