Comp Sci Principles Web Site help

Wanna make a nice website? Head over to wix.com/sanjit1 to get 50% off on the first year of your subscription. JK Lmao.
Since you are in CSP right now, you might need some help making your first website. The first thing you wanna do is make a github account. It may prompt you to make a new repository, name the repository "yourusername.github.io": Eg: sanjit1.github.io The next thing you will need to do is to make a home page:
Since you want it to be the first thing that pops up when someone opens your website, you want to name it index.html
Steps:

  • Click on the Create new file button
  • In the name your file section, name the file: index.html
  • Get ready to write your first html code!

In the file editor start by adding a doctype tag, <!DOCTYPE html>. This tells the browser that your file is an HTML file, which helps the browser render your file.
The next line of code you will need is an html tag: <html> </html> HTML is a markup language, so it works with tags. Most tags have to be closed with a /, so that the browser knows where to stop rendering the tag. For example: <body> </body> However some tags do not need a closing tag, For example: <img>. Inside the html tag, all the html code will reside. After that you need to make a head tag and a body tag. Both need to be closed. So far you should have something like this: 

                
<!DOCTYPE html> 

<html> 
    <head> 

    </head> 

    <body> 

    </body> 
<html>

In your head tag you can add a title and CSS. We will go over what CSS pretty soon. You can add this into your head tag for now <title>Welcome to my Home page!</title>
Next you want to focus on your body: inside the body tag add this code <h1>Welcome to my Home!<h1> Now you should have:

                
<!DOCTYPE html>
<html> 
    <head> 
        <title>Welcome to my Home Page!</title>
    </head> 

    <body> 
        <h1>Welcome Home<h1>
    </body> 
<html>

Fuck yea you just made a website. Save your Changes by clicking the commit button.
Now it is time to deploy your website to the interwebs. Here are the steps you need to follow to deploy your website:

  • Go to the settings tab of the repository:

  • Scroll down till you see Github pages, and click on the None. Then click on master. You should see something like this:

  • Wait until it says Your site is published at: ... (You might have to reload a couple of times)
  • Click on the link it says the website it is published at, and you have made a fucking website! Hell ya

Now you need to make some improvements to your website: Here are a couple of things that you need to add to your website:

  • An Image on the home Page.
    To add a picture on your home page, Download a picture of a house from the internet, and add it to your website by dragging the download onto the main page of your repository on github.

    Click on commit Changes. Now you have an image file on your website. Next you need to figure out where to place it. Go to the index.html file and start editing it. Add this line of code to your file: <img src="./image.png"> WARNING: replace the image.png with the filename of the image you uploaded. In my case it was binoculars.png. This is case-sensitive. Click on commit.
  • Make a portfolio page and an about me page. Click on new file, and write html code for it below is some sample code. Make sure you name the file appropriately: The sample code has the name of the file you should use.
    Here is some code if you are too lazy:
    File name:aboutme.html 
    
<!DOCTYPE html>
<html> 
    <head> 
        <title>About Me</title>
    </head> 

    <body> 
        <h1>This page is all about me<h1>
        <img src="myface.png">
        <h2>My Face is above</h2>
        <p>I hate coding!</>
    </body> 
<html>
    File name:portfolio.html 
    
<!DOCTYPE html>
<html> 
    <head> 
        <title>CSP Portfolio</title>
    </head> 

    <body> 
        <center>
        <h1>This page is for all the stuff we learn in CSPe<h1>
        <img src="code.png">
        <h2>That is an image of my code</h2>
        <p>I made this project with my partner: Sanjit</>
        </center>
    </body> 
<html>

    Before we move on I want to clarify the tags used in the code above: Here are the w3School links for them:

    • H tag(h1,h2 etc)
    • Image tag
    • P tag
    • Center tag

    Now that you have the pages in place, you need to add navigation, so that you can navigate to your portfolio and about me page from your home page etc. Attatch this code right inside the body tags of each page. 

    
<a href="/index.html">Home</a>
<a href="/aboutme.html">About me</a>
<a href="/portfolio.html">CSP Portfolio</a>
